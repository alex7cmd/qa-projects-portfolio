Proyecto 01

API Testing · Automatización

Urban Grocers: automatización de pruebas de API

Problema
Validar que la creación de kits de productos funcionara correctamente y manejara errores de entrada de datos.
Qué hice
Diseñé y ejecuté pruebas automatizadas con Python y Pytest para verificar el campo name mediante escenarios positivos, negativos y valores límite.
Resultado o aprendizaje
Fortalecí mis bases en testing de APIs, autenticación y reglas de negocio directamente en el backend.
  • Python
  • Pytest
  • API REST
  • JSON
